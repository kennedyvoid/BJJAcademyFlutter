# Jiupedia — Flutter

Aplicativo Flutter para iOS e Android com posições permitidas por faixa, técnicas proibidas por faixa, busca, favoritos, pontuação, tema claro/escuro e ícone próprio.

## Como executar

```bash
flutter create .
flutter pub get
dart run flutter_launcher_icons
flutter run
```

## iOS / Xcode

```bash
open ios/Runner.xcworkspace
```

## Android APK

```bash
flutter build apk --release
```

## Observação sobre imagens

As imagens ficam em `assets/images/`. Para publicação comercial, substitua por fotos próprias ou licenciadas corretamente para cada posição.
